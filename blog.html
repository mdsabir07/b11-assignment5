Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in the DOM. Such as- select by id(getElementById), select by class name(getElementsByClassName), select with tag name (getElementsByTagName), select with name (getElmentsByName), select with selector (querySeletor, querySeletorAll), parentNodes, childNodes, etc.

  2. innerText: display all content except the HTML tags, class, attributes, etc. But hidden content not showing here!

Question-3: What is event delegation in the DOM?

Event delegation allows to add event listener to parent element instead of multiple child elements. It's help to improvde the performance by reducing the numbers of event listeners. It's leveragin event bubbling and using the event.target property its determine which child element trigered by event.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism of DOM. When its triggered an elements, "bubbles up" from the target element to root documents. Basically when we click the button, the event is trigered the button element itself. Than it's bubbles up to the child element. Next it's bubbles up to the parent element (div). In this process it's continue bubbling up until reachs the root documents.
